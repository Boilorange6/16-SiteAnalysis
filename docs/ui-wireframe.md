## Component Breakdown

### Left Sidebar (320px fixed)

1. **Address Search** - 텍스트 입력 + 자동완성(Mapbox Geocoding)
2. **Range Selector** - 1km / 2km / 3km 라디오 버튼
3. **POI Toggle** - 카테고리별 체크박스 (지하철/학교/공원/산)
4. **Apartment Section** - 자동 로드된 아파트 목록 + 수동 추가 버튼
5. **PPT Preview** - 썸네일 미리보기 (4~6장)
6. **Download Button** - PPT 다운로드 (Primary CTA)

### Main Area (Remaining Width)

- Mapbox GL JS 전체 화면 위성지도
- 오버레이 레이어: POI 마커, 아파트 마커, 거리선, 반경 원
- 클릭 시 팝업: 상세 정보 표시

## Interaction Spec

| Action | Response |
|--------|----------|
| 주소 입력 | 자동완성 드롭다운 → 선택 시 지도 이동 |
| 반경 변경 | 지도 위 반경 원 업데이트 + POI 재로드 |
| POI 토글 | 해당 레이어 표시/숨김 (애니메이션 fade) |
| 마커 클릭 | 팝업: 이름, 거리, 상세정보 |
| Export PPT | 로딩 스피너 → 자동 다운로드 |
