# 2단계 디자인 언어 — 기존 보고서(260311 사이트현황) 분석

원본: `docs/260311_목동2단지_현황.pptx` (9슬라이드, 16:9, dA architecture group) — **레포에 커밋 금지** (실제 프로젝트 정보 포함). 본 문서가 디자인 토큰의 소스.

## 타이포그래피

- 주력: **Noto Sans CJK KR** (Regular 본문 / Medium / Bold 타이틀 / Black 강조)
- 보조: **Pretendard** (Medium / SemiBold / ExtraLight — 메타·숫자 계열)
- 표지 타이틀: 초대형 Bold 화이트 (슬라이드 폭의 ~70%), 아이브로우는 자간 넓은 소형
- 두 폰트 모두 무료(OFL) — next/font + PPT fontFace 적용 가능

## 슬라이드 유형별 문법

### A. 표지 (slide 1)
- 거의 검정(#1a1a1a급) 배경, 우측에 얇은 흰 테두리 사각 장식 2-3개(오프셋 배치)
- 좌상단: 아이브로우 2줄(프로젝트명 / 문서종류 — 자간 넓게)
- 좌하단: 초대형 타이틀 + 그 아래 메타 행 `PART10 / UAD3 / 2026. 03. 12. 목` (슬래시 구분)
- 우상단: 로고

### B. 백색 정보 슬라이드 (slide 2 프로젝트 소개, slide 9 결정도)
- 흰 배경 + 중앙 상단 제목(좌우 수평선 플랭크) + 라운드 대형 외곽 프레임
- **검정 헤더 표**: 행 라벨(위치/구역면적/용적률/높이/세대수/용도) + 값, **핵심 수치는 빨강 강조**
- 라운드 테두리 카드(일정 등), 회색 패널에 ■ 불릿 계층 요약
- → 앱의 "팩트 시트" 문법으로 채택 (2단계 핵심)

### C. 지도 분석 슬라이드 (slides 3·5·6·7·8 — 본론)
- **풀블리드 흑백(탈채도+어둡게) 위성지도** — 현재 앱의 컬러 위성+네이비 오버레이와 다름
- 좌상단: 볼드 화이트 섹션 타이틀(예: 녹지·자연경관) + 흰 서브라벨
- 노선: **점선(dashed) 폴리라인** + 역 위치 도트 (노선색: 노랑/보라/초록/파랑)
- 폴리곤: 대상지 **빨강**, 비교 단지 베이지, 특수 강조 골드/블루
- 콜아웃: 흰 리더라인 + **미니 데이터표** (준공/규모/용적률/세대수 — 대상지는 빨강 헤더, 나머지 흰/검정)
- **인사이트 카드**: 우측(하단) 라운드 검정 카드 + 흰 볼드 3-5줄 요약 ← 각 카테고리의 "결론" 위치
- 범례: 좌하단 색 도트+라벨
- 지명: 도로 흰색, 산 초록(+높이m), 수계 하늘색
- slide 4: 검정 바탕 2×2 항공사진 그리드 (앱 범위 외 — 수동 삽입 영역으로 남김)

## 현재 앱과의 갭 (2단계 작업 항목으로 번역)

| 항목 | 현재 앱 | 목표 |
|---|---|---|
| 폰트 | 시스템/기존 폰트 | Noto Sans CJK KR + Pretendard (웹+PPT 동시) |
| 지도 톤 | 컬러 위성 + 네이비 반투명 | 흑백 탈채도 + 어둡게 (오버레이 색이 주인공) |
| 노선 | 실선 | 점선 + 역 도트 |
| 대상지 | 마커+링 | 빨강 폴리곤(대상지) 중심 |
| 카테고리 결론 | 없음/점수 위주 | 라운드 검정 인사이트 카드 (팩트 요약) |
| 콜아웃 | 자체 스타일 카드 | 리더라인+미니 데이터표 (빨강 헤더 = 대상지) |
| 표지 | 위성 배경+타이틀 | 흑색 배경+초대형 타이틀+테두리 장식 |
| 개요/출처 | 다크 패널 | 백색 정보 슬라이드(검정 헤더 표+빨강 강조) |
| 강조색 | 파랑 계열 | **빨강**(수치 강조·대상지), 노선색은 실색 |

## 슬라이드 구성 시사점 (팩트 중심 재편과 결합)

원본 보고서 구성: 표지 → 프로젝트 소개(팩트 표) → 광역 입지(지도) → 항공사진 → 카테고리 분석(지도+인사이트 카드) ×N → 단지 현황(콜아웃 표) → 결정도(백색 요약).
→ 앱 매핑: 표지 → **공모개요 팩트 시트(신규, 사용자 입력+자동 팩트)** → 입지 종합(지도) → 교통/교육/자연/주거 (지도+팩트 인사이트 카드) → 아파트 콜아웃(미니 데이터표) → 데이터 출처(백색).
- 점수 대시보드 슬라이드는 기본 제외(로드맵 확정사항), 종합 의견 카드로 대체
