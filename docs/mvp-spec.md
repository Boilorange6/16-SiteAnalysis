# Site Analysis PPT Generator - MVP Specification

## Product Overview

**목적:** 부동산 사이트 분석 보고서(PPT)를 자동 생성하는 웹 애플리케이션
**핵심 가치:** 수작업 PPT 제작 시간을 90% 이상 단축

## MVP Scope

### Core Features (P0)

#### 1. 위치 지정 및 위성지도 렌더링
- 사용자가 주소 또는 좌표로 분석 대상 위치 입력
- Mapbox GL JS 기반 위성지도 렌더링
- 줌 레벨/범위 조정 가능

#### 2. POI(Point of Interest) 오버레이
- **지하철역:** 노선별 색상 아이콘 + 역명 라벨
- **학교:** 초/중/고 구분 아이콘
- **공원:** 녹색 영역 폴리곤
- **산:** 등고선 영역 또는 아이콘

#### 3. 아파트 분양 정보 오버레이
- 사용자 지정 반경 내 분양 아파트 표시
- 아파트명, 세대수, 분양가 텍스트 라벨
- 거리 표시선(대상지~아파트)

#### 4. PPT Export
- 위성지도 + 오버레이를 고해상도 이미지로 캡처
- PptxGenJS로 슬라이드 생성
- 슬라이드 구성:
  - 표지 (프로젝트명, 날짜, 위치)
  - 전체 현황도 (위성지도 + 전체 POI)
  - 교통 분석 (지하철 중심)
  - 교육 환경 (학교 중심)
  - 자연 환경 (공원/산 중심)
  - 분양 현황 (아파트 중심)

### Out of Scope (MVP 제외)
- 사용자 인증/로그인
- 분석 결과 저장/히스토리
- 다중 사용자 협업
- 커스텀 아이콘 업로드
- AI 기반 분석 코멘트 자동 생성

## User Flow

```
1. 웹 접속
2. 분석 대상 주소 입력 (검색/직접 입력)
3. 지도에서 중심점 확인 및 분석 반경 설정 (기본 2km)
4. POI 카테고리별 표시 ON/OFF 토글
5. 아파트 분양 정보 자동 로드 + 수동 추가 가능
6. 미리보기에서 레이아웃 확인
7. "PPT 다운로드" 버튼 클릭
8. PPT 파일 다운로드
```

## Data Sources

| 데이터 | 소스 | 비고 |
|--------|------|------|
| 위성지도 | Mapbox Satellite | API Key 필요 |
| 지하철역 | 공공데이터포털 (국토교통부) | 전국 지하철역 좌표 |
| 학교 | 학교알리미 API / 공공데이터 | 초/중/고 위치 |
| 공원 | 공공데이터포털 (도시공원) | 공원 경계 폴리곤 |
| 산 | 국토지리정보원 | 산 위치/경계 |
| 아파트 분양 | 공공데이터포털 (분양정보) | 분양가/세대수 포함 |

## Non-Functional Requirements

- **성능:** 지도 로딩 3초 이내, PPT 생성 10초 이내
- **브라우저:** Chrome/Edge 최신 2개 버전
- **해상도:** 1920x1080 기준 최적화, 반응형 불필요(MVP)
- **PPT 품질:** 16:9 와이드, 지도 이미지 최소 1920x1080px
