# 개발팀 킥오프 보완계획

개발팀 준비 상태는 양호하며, 보완이 필요한 구체 항목은 다음과 같습니다:

1. **지도 타일 소스 선정**: 위성지도 배경용으로 Kakao Maps 또는 Naver Maps Static API 중 어떤 것을 쓸지 확정 필요.
2. **청와대 바운딩박스**: 청와대 좌표(37.5866, 126.9748) 중심 반경 3km 바운딩박스 계산 및 오차 검증.
3. **아파트 분양 데이터 마커 표현 방식**: 분양 아파트 데이터의 마커 렌더링, 라벨 겹침 방지 처리 로직 구현.
4. **PPT 레이어 처리**: 전체뷰 1장과 카테고리별 레이어 오버레이 분리뷰 렌더링 방식 확정.

이 보완계획을 바탕으로 청와대 3km 반경 테스트 스크립트 및 분석 결과물을 준비하겠습니다.
